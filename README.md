# zeemee-demo-ios
A working version of react-native with relay and graphql.


### Clone this repo

```
$ git clone https://github.com/zeemee/zeemee-demo-ios.git
$ cd zeemee-demo-ios
```

### Create an account at https://www.zeemee.com/students/sign_up

### Run the Native App
```
$ open ios/fix.xcodeproj
```
